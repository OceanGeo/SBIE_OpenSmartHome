# SBIE_OpenSmartHome
SBIE_OpenSmartHome – Έξυπνο Σπίτι Ανοιχτού Κώδικα για Ενεργειακή Εξοικονόμηση και Προσβασιμότητα
1. Τίτλος Έργου
OpenSmartHome – Έξυπνο Σπίτι Ανοιχτού Κώδικα για Ενεργειακή Εξοικονόμηση και Προσβασιμότητα
Έργο για τον 8ο Πανελλήνιο Διαγωνισμό Ανοιχτών Τεχνολογιών στην Εκπαίδευση.
2. Περίληψη Έργου
Το έργο αφορά τη σχεδίαση ενός έξυπνου συστήματος κατοικίας με χρήση PLC και Arduino. Στόχος είναι η ενεργειακή αποδοτικότητα, η ασφάλεια και η κοινωνική συμπερίληψη.

3. Κοινωνική Διάσταση
Το έργο συμβάλλει:
- Στη μείωση ενεργειακής φτώχειας.
- Στην υποστήριξη ηλικιωμένων και ΑμεΑ.
- Στην αύξηση της ασφάλειας κατοικιών.
- Στη διάδοση ανοιχτών τεχνολογιών.

4. Εκπαιδευτική – Διδακτική Διάσταση
Οι καταρτιζόμενοι ΣΑΕΚ συμμετέχουν σε:
- Σχεδίαση αυτοματισμών.
- Προγραμματισμό PLC και Arduino.
- Ανάπτυξη IoT εφαρμογών.
- Ομαδική εργασία και τεχνική τεκμηρίωση.

5. Αναλυτική Τεχνική Περιγραφή
Το Arduino συλλέγει δεδομένα αισθητήρων και επικοινωνεί με PLC. Το PLC ελέγχει φωτισμό, θέρμανση, ασφάλεια και ηλεκτρικές συσκευές.

6. Αισθητήρες
- Θερμοκρασίας και υγρασίας.
- Κίνησης (PIR).
- Καπνού και αερίου.
- Κατανάλωσης ρεύματος (ACS712).
- Φωτεινότητας.
- Ανοίγματος θυρών/παραθύρων.
- Πλημμύρας/διαρροής νερού.
- Θορύβου.
- Ποιότητας αέρα.
- RFID / βιομετρικός έλεγχος πρόσβασης.

7. Ενεργοποιητές
- Ρελέ φωτισμού.
- Έξυπνες πρίζες.
- Ηλεκτροβάνες.
- Συστήματα συναγερμού.
- Σερβοκινητήρες.
- Οθόνες HMI.

8. Λογισμικό και Ανοιχτές Τεχνολογίες
- Home Assistant.
- Node-RED.
- MQTT.
- OpenPLC.
- Web dashboards.
Όλες οι εφαρμογές βασίζονται σε ανοιχτό λογισμικό.

9. Καινοτομία
Το έργο συνδυάζει βιομηχανικούς αυτοματισμούς με IoT και κοινωνικό προσανατολισμό.

10. Αναμενόμενα Αποτελέσματα
- Ανάπτυξη σύγχρονων τεχνικών δεξιοτήτων.
- Μείωση κατανάλωσης ενέργειας.
- Δημιουργία πρότυπου μοντέλου Smart Home ανοικτού κώδικα.
